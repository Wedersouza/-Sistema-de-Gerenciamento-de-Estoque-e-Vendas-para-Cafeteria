# Sistema de Gerenciamento de Estoque e Vendas para Cafeteria

CSI606-2026-01 - Remoto - Proposta de Trabalho Final
Discente: Weder Araujo de Souza

## Resumo

Este trabalho propõe o desenvolvimento de um Sistema de Gerenciamento de Estoque e Vendas para Cafeteria, com o objetivo de auxiliar na administração das operações de uma cafeteria de pequeno ou médio porte. O sistema permitirá o gerenciamento de produtos, estoque, vendas, clientes, fornecedores e usuários, oferecendo uma interface simples, organizada e intuitiva. Além disso, disponibilizará um painel administrativo com indicadores das principais operações, contribuindo para o controle do estabelecimento e apoiando a tomada de decisões.

O sistema também contará com funcionalidades de autenticação, cadastro de novos usuários, gerenciamento do perfil do usuário autenticado e alteração de senha, proporcionando maior segurança e facilidade de utilização.

## 1. Tema

O trabalho final tem como tema o desenvolvimento de um Sistema de Gerenciamento de Estoque e Vendas para Cafeteria, voltado ao gerenciamento das operações comerciais e ao controle de estoque do estabelecimento.

## 2. Escopo

Este projeto terá as seguintes funcionalidades:

- Autenticação de usuários por meio de login;
- Cadastro de novos usuários;
- Cadastro e gerenciamento de usuários, com perfis de acesso (Administrador e Funcionário);
- Cadastro de produtos (cafés, bebidas, salgados, sobremesas e demais itens);
- Controle de estoque integrado ao cadastro de produtos, permitindo registrar entradas e saídas de mercadorias;
- Cadastro de clientes;
- Cadastro de fornecedores;
- Registro de vendas com atualização automática da quantidade disponível em estoque;
- Controle de preços e quantidades dos produtos;
- Consulta ao histórico de vendas;
- Dashboard administrativo com indicadores das principais informações do sistema, como:
  - vendas realizadas;
  - produtos mais vendidos;
  - produtos com baixo estoque;
  - últimas vendas;
- Gerenciamento do perfil do usuário autenticado;
- Atualização das informações pessoais do usuário;
- Alteração de senha pelo usuário autenticado.

## 3. Restrições

Neste trabalho não serão considerados:

- Integração com sistemas bancários ou meios de pagamento on-line;
- Emissão de documentos fiscais;
- Programa de fidelidade;
- Aplicativo para dispositivos móveis;
- Serviço de delivery com geolocalização;
- Controle financeiro e contábil avançado;
- Integração com marketplaces ou plataformas externas;
- Controle de múltiplas unidades ou filiais.

## 4. Protótipo

Protótipos para as seguintes páginas foram elaborados: Tela de Login, Tela de Cadastro de Usuário, Dashboard Administrativo, Tela de Produtos (cadastro e controle de estoque integrado), Tela de Vendas, Tela de Clientes, Tela de Fornecedores, Tela de Usuários e Tela Meu Perfil (com Informações Pessoais e Alteração de Senha). Os protótipos podem ser encontrados no documento "Protótipos.pdf", anexado a esta proposta de trabalho.
