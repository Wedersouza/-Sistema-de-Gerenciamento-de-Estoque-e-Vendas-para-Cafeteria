# -Sistema-de-Gerenciamento-de-Estoque-e-Vendas-para-Cafeteria

Resumo

O presente trabalho propõe o desenvolvimento de um Sistema de Gerenciamento de Estoque e Vendas para Cafeteria, com o objetivo de auxiliar no controle de produtos, estoque, vendas e atendimento ao cliente em um estabelecimento comercial. A aplicação permitirá organizar o cadastro de produtos, monitorar entradas e saídas de estoque, registrar vendas e gerar informações gerenciais básicas para apoio à tomada de decisão. O sistema busca melhorar a eficiência operacional, reduzir perdas e otimizar a administração de cafeterias de pequeno e médio porte.

1. Tema
O trabalho final tem como tema o desenvolvimento de um Sistema de Gerenciamento de Estoque e Vendas para Cafeteria, voltado ao controle operacional e comercial do estabelecimento.


2. Escopo
Este projeto terá as seguintes funcionalidades:

Cadastro de produtos (cafés, bebidas, salgados, sobremesas e demais itens);
Controle de estoque com registro de entrada e saída de produtos;
Atualização automática de estoque a partir das vendas realizadas;
Cadastro de fornecedores;
Registro de vendas e pedidos;
Cadastro de clientes;
Controle de preços e quantidades disponíveis;
Relatórios simples de vendas, produtos mais vendidos e nível de estoque;
Consulta de histórico de vendas;
Painel administrativo para acompanhamento das operações.


3. Restrições
Neste trabalho não serão considerados:

Integração com sistemas bancários ou pagamentos online;
Delivery com geolocalização;
Aplicativo mobile;
Programa de fidelidade;
Emissão fiscal oficial;
Controle contábil avançado;
Integração com marketplaces ou aplicativos externos.


4. Protótipo
Protótipos para as páginas principais foram elaborados, contemplando:

Página inicial (dashboard administrativo);
Página de cadastro de produtos;
Página de controle de estoque;
Página de cadastro de fornecedores;
Página de registro de vendas e pedidos;
Página de cadastro de clientes;
Página de relatórios gerenciais.
